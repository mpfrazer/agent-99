## Design Summary

| Concern | Decision |
|---|---|
| Frontend | Next.js 14 (App Router), TypeScript, Tailwind CSS |
| Backend | FastAPI (Python), extends existing agent99 runner |
| Streaming | Server-Sent Events (SSE) — one-way, simpler than WebSockets |
| Run tracking | In-memory `{run_id: RunState}` dict; survives navigation via context |
| Run persistence | `~/.agent99/runs/YYYY-MM-DD-HH-MM-SS-<agent>.md` (YAML frontmatter + body) |
| Auth | Single password, bcrypt hash in `~/.agent99/config.yaml`, JWT httpOnly cookie, 30-day expiry |
| Install | `install.py` (cross-platform Python), `start.py` (launches both servers) |
| Dev proxy | next.config.js rewrites `/api/*` → `http://localhost:8000/api/*` |

## API Endpoints

### Auth
- `POST /api/auth/login` — `{password}` → sets httpOnly JWT cookie + `agent99_logged_in` flag cookie
- `POST /api/auth/logout` — clears cookies
- `GET /api/auth/me` — `{authenticated: bool}` (used by Next.js middleware)

### Agents
- `GET /api/agents` — list all agent YAMLs
- `GET /api/agents/{name}` — get agent config
- `POST /api/agents` — create new agent
- `PUT /api/agents/{name}` — update agent
- `DELETE /api/agents/{name}` — delete agent

### Runs
- `POST /api/runs` — start run → `{run_id}`
- `GET /api/runs` — list runs (from files + active in-memory)
- `GET /api/runs/{id}` — run detail
- `GET /api/runs/{id}/stream` — SSE stream
- `DELETE /api/runs/{id}` — cancel run

## Auth flow

1. User visits any protected page → Next.js middleware checks `agent99_logged_in` cookie
2. If missing → redirect to `/login`
3. Login page → `POST /api/auth/login` → FastAPI verifies password, sets cookies
4. All API calls include the httpOnly `agent99_session` JWT cookie automatically
5. FastAPI `require_auth` dependency decodes JWT, returns 401 if invalid
6. Logout → `POST /api/auth/logout` → clears both cookies

## Active runs tracking

- `ActiveRunsContext` (React Context) holds `Map<runId, RunSummary>`
- On run start: add to context immediately
- `StreamingOutput` opens SSE connection; on done/error/cancelled → update context
- `ActiveRunsBar` (sticky bottom) reads context, shows spinner / status / cancel button
- On page reload: `GET /api/runs?status=running` rehydrates context

## Install flow

```
git clone https://...
python install.py
  → checks Python 3.11+, Node 18+
  → creates .venv, pip install -e ".[web]"
  → npm install && npm run build in web/
  → creates ~/.agent99/, prompts for password
  → saves ~/.agent99/config.yaml

python start.py
  → starts FastAPI on :8000
  → starts Next.js on :3000
  → opens http://localhost:3000 in browser
  → Ctrl+C shuts both down
```
